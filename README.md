# Numerology Compass

A multilingual (Romanian/English/Russian) numerology web application using the Pythagorean system.

## Features

- Life Path, Destiny, Soul Urge, and Personality number calculations
- Compatibility analysis between two people
- Karmic debt and lessons identification
- Personal cycles (day/month/year)
- Daily and monthly predictions
- Advanced tools: name analysis, year forecast, compatibility matrix

## Tech Stack

- React 18 + TypeScript
- Vite 6 (SWC)
- shadcn/ui + Tailwind CSS
- React Router + React Query
- Express + PostgreSQL (backend)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Backend

```bash
cd server
npm install
npm run dev
```
