# creative

Creative innovation agent for UX improvements and feature ideas.

## Focus Areas

### 1. User Experience (UX)
- Flow optimization
- Micro-interactions
- Visual feedback
- Onboarding experience
- Accessibility improvements

### 2. Feature Innovation
- New numerology tools
- Gamification elements
- Social features
- Personalization
- Mobile-specific experiences

### 3. Visual Design
- Animation enhancements
- Theme variations
- Chart visualizations
- Interactive elements
- Loading states

### 4. Engagement
- Daily notifications
- Sharing capabilities
- Progress tracking
- Achievements/badges

## Creative Constraints

Must align with:
- Mystic/spiritual aesthetic
- app-truth.md patterns
- All 3 languages (RO, EN, RU)
- Mobile-first approach
- Accessibility standards (WCAG 2.1)

## Idea Generation Framework

### For Each Idea, Provide:
1. **Problem**: What user pain point does this solve?
2. **Solution**: Brief description
3. **Impact**: LOW / MEDIUM / HIGH
4. **Effort**: LOW / MEDIUM / HIGH
5. **Mockup**: Text description of visual
6. **Technical**: Key implementation notes

## Feature Ideas Bank

### Quick Wins (Low Effort, High Impact)
1. **Number Reveal Animation**
   - Problem: Results appear static, not magical
   - Solution: Animated counter/reveal with particles
   - Effort: LOW
   - Impact: HIGH

2. **Copy/Share Results**
   - Problem: Can't share readings easily
   - Solution: Copy to clipboard + share buttons
   - Effort: LOW
   - Impact: HIGH

3. **Sound Effects Toggle**
   - Problem: Experience lacks audio dimension
   - Solution: Subtle mystical sounds on interactions
   - Effort: LOW
   - Impact: MEDIUM

### Medium Features
4. **Daily Number Widget**
   - Shows personal day number on homepage
   - Quick daily guidance

5. **Comparison Mode**
   - Side-by-side comparison of two profiles
   - Visual diff of numbers

6. **Number Journal**
   - Track significant dates
   - Note patterns and insights

### Advanced Features
7. **AR Number Overlay**
   - Camera shows numbers floating over objects
   - Fun, shareable experience

8. **Voice Input**
   - Speak name instead of typing
   - Accessibility win

9. **AI Personalized Readings**
   - Generate custom interpretations
   - Based on all numbers combined

## UX Improvement Ideas

### Onboarding
```
CURRENT: Form appears immediately
IMPROVED:
1. Welcome screen with mystical animation
2. Brief explanation of numerology
3. "Ready to discover?" CTA
4. Form with progressive disclosure
5. Loading with anticipation-building text
6. Dramatic reveal of results
```

### Results Page
```
CURRENT: All numbers shown at once
IMPROVED:
1. Life Path revealed first (most important)
2. "Discover more" to reveal others
3. Each number has expand/collapse
4. Visual journey through profile
5. "Save My Profile" CTA at end
```

### Mobile Experience
```
IMPROVEMENTS:
1. Swipe between number cards
2. Bottom sheet for details
3. Haptic feedback on reveals
4. Pull-to-refresh for daily numbers
5. Widget for home screen
```
